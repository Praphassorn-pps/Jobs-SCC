## 🎨 **Design System**

### **Color Palette**
```
Primary Colors:
- LINE Green: #06C755
- LINE Dark Green: #1DB146  
- White: #FFFFFF
- Dark: #1A1A1A

Secondary Colors:
- Gray 50: #F9FAFB
- Gray 100: #F3F4F6
- Gray 500: #6B7280
- Gray 900: #111827

Status Colors:
- Success: #10B981 
- Warning: #F59E0B
- Error: #EF4444
- Info: #3B82F6
```

### **Typography**
```
Font Family: Sarabun (Thai), Inter (English)
Headings: 600-700 weight
Body: 400 weight
Captions: 300 weight
```

## 🎯 **User Flow Diagrams**

### **User Registration Flow**
```
LINE OA → Rich Menu Click → Web App → LINE Login → PDPA Consent → Profile Complete → Dashboard
```

### **Job Application Flow**  
```
Job Notification → View Details → Apply → Upload Documents → Answer Questions → Submit → Confirmation
```

### **Interview Process Flow**
```
HR Select → Send Invitation → User Confirm → Interview Date → Record Result → Send Notification
```
